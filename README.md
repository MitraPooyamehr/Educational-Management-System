# 🎓 Educational Management System (Python & SQL)

Welcome to my repository! I am an Economic Analyst and Data Enthusiast transitioning my theoretical knowledge into practical, data-driven software solutions. 

This project is a fully functional **Educational Management System** with a Graphical User Interface (GUI) built from scratch. It demonstrates the integration of backend relational database management with frontend user interactions using a professional N-Tier layered architecture.

## 🚀 Key Features
- **Relational Database Architecture:** Seamlessly connects multiple entities (Students, Teachers, Courses, Departments). Schema is available in `Database_Schema.sql`.
- **Full CRUD Operations:** Add, Read, Update, and Delete records effortlessly with live search filtering.
- **Score & Certification Management:** Tracks student grades, calculates pass/fail statuses, and manages professional IT certifications.
- **Automated Reporting:** - 1-click export of data to **Excel**.
  - Automated PDF generation for **Student ID Cards**.
  - Direct printing capabilities for **Official Contracts**.

## 🛠️ Technology Stack
- **Programming Language:** Python
- **GUI Framework:** Tkinter
- **Database:** SQL (Relational Database Management)
- **Architecture:** N-Tier Architecture (BLL, DAL, Model, UI)
